# Application Form

Одностраничная форма заявки, реализованная на чистом HTML, CSS и Vanilla JavaScript.

## Описание
Проект демонстрирует умение работать с семантикой форм, UI-состояниями и базовой клиентской логикой без использования фреймворков.

Форма включает валидацию полей, обработку ошибок и success-состояние после успешной отправки.

## Стек
- HTML5 (семантика форм)
- CSS3 (mobile-first, focus / error / success states)
- Vanilla JavaScript (валидация, управление состояниями)

## Реализовано
- Семантическая структура: `form`, `fieldset`, `legend`, `label`
- Mobile-first layout
- UI-состояния:
  - focus
  - error
  - disabled
  - success
- Пользовательская валидация (`novalidate`)
- Универсальная логика проверки полей
- Отдельная логика для checkbox
- Отображение success-сообщения без перезагрузки страницы

## Цель проекта
Показать базовый фронтенд-фундамент:
- чистую архитектуру формы
- контроль UI-состояний
- осознанную работу с DOM и событиями
